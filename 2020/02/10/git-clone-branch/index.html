git clone 指定分支拉代码

2020-02-10 13:07
#github

git clone 不指定分支

1

git clone  http://10.1.1.11/service/tmall-service.git

git clone 指定分支

1

git clone -b dev_jk http://10.1.1.11/service/tmall-service.git

命令中：多了一个  -b dev-jk,这个dev_jk就是分支，http://10.1.1.11/service/tmall -service.git为源码的仓库地址

Feh Open SVG

üïì 2020-04-01 21:58

#linux

Without the –conversion-timeout part, I run into this error: feh: No Imlib2 loader for that file format

As suggested in the Arch Linux Wiki, I open SVG images like this:

feh --conversion-timeout 1 file.svg

This will create a “feh” theme, which will be loaded by default and which will automatically apply the conversion-timeout option.

echo "feh --conversion-timeout 5" > ~/.config/feh/themes
